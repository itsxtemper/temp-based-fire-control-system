
# Temperature Based Fire Control System

Components Required:

    1. DHT11 (Temperature and Humidity Sensor)
    2. 16x2 Liquid Crystal Display (without I2C)
    3. Arduino Uno R3
    4. Relay Module (JQC3F-5VDC-C)
    5. Buzzer
    6. Potentiometer (8-10K)
    7. Red and Green LEDs (If u want to)
    8. Breadboard,
    9. Wires (as required)
